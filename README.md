# PullFrame

PullFrame is a simple macOS app for saving supported public media links through a clean local interface.

## Download

Download the latest version from the Releases page:
https://github.com/Fat7i2017/pullframe-releases/releases/latest

Download:
PullFrame.dmg

## Install on macOS

1. Download PullFrame.dmg
2. Open the DMG file
3. Drag PullFrame into Applications
4. Open PullFrame from Applications
5. After installation, eject the installer
6. Delete PullFrame.dmg from Downloads if you want

## Updates

Open PullFrame and choose:
Check for Updates

If a newer version is available, PullFrame will open the latest download page.

## Requirements

PullFrame currently requires these local tools:
- yt-dlp
- FFmpeg

They are used locally on your Mac to fetch and process media links.

## Safety and legal note

Use PullFrame only for content you own, have permission to download, or are legally allowed to save.

PullFrame is not affiliated with yt-dlp, FFmpeg, YouTube, Instagram, TikTok, X/Twitter, Vimeo, Facebook, or any supported platform.
